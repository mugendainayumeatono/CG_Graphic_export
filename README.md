# CrossGate Graphic Exporter (魔力宝贝图档导出工具)

这是一个基于 Python 和 PySide6 (Qt) 开发的带有图形界面的《魔力宝贝》（CrossGate）客户端底层图档解析与导出工具。
本项目通过解析客户端原始的二进制格式数据（如 `GraphicInfo*.bin`，`Graphic*.bin`，`palet_*.cgp`），直接还原并渲染出带有透明通道的 PNG 图像。

## ✨ 主要功能

1. **图档预览**：支持读取底层的 `Graphic_*.bin` 压缩数据，结合调色板实时渲染预览出原生图像。
2. **多调色板支持**：可以根据不同的游戏时间/场景动态切换调色板（例如白天、黄昏、夜晚），实时查看不同配色下的表现。
3. **单张/批量导出**：
   - 选定心仪的图档后，点击导出单张图片。
   - 也可以一键无脑将整个版本的图档批量转化为 PNG 图片。
4. **透明通道导出**：彻底解决早年截图有黑边或多余背景色的烦恼，导出的 PNG 文件将未绘画像素和标记为透明的像素转为完整的 Alpha 透明区域。

## 🛠️ 安装与运行环境

本项目推荐使用 [uv](https://github.com/astral-sh/uv) 进行极速环境依赖管理。

1. **安装 uv** (如果您还没有安装过):
   - Windows (PowerShell): `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`
   - 更多安装方法见官网。

2. **拉取代码后运行项目**:
   在 `CrossGate_Graphic_exporter` 目录中执行以下命令，`uv` 将会自动安装所缺少的依赖 (`pyside6`, `pillow`, `numpy`) 并启动图形界面：
   ```bash
   cd CrossGate_Graphic_exporter
   uv run src/main.py
   ```

## 🎮 使用方法

1. **准备原始游戏数据**：
   此导出器需要直接从游戏客户端目录读取数据。请准备好带有以下两个文件夹的父级目录：
   - `bin/` 目录：包含所有的 `GraphicInfo*.bin` 和 `Graphic*.bin` 文件。
   - `pal/` 目录：包含所有的调色板文件，如 `palet_00.cgp` 等。

2. **启动软件与加载数据**：
   - 工具启动时，会尝试**自动扫描**当前所在的目录以及上级目录中是否包含合法的 `bin/`。如果存在，系统会自动加载并列出所有可供导出的图档。
   - 如果自动扫描失败，请在界面的顶部点击 **"Browse Base Dir"** 手动选择同时包含 `bin` 和 `pal` 文件夹的上级目录。

3. **预览操作**：
   - 在左侧列表中选中目标图档（展示格式为 `Serial: 编号 | Index: 索引`），右侧即刻刷新出预览。
   - 您可通过上方的 `Palette` 下拉框切换配色。

4. **导出操作**：
   - **Export Preview**：只导出当前在屏幕右侧显示的这张图，并且会保留当前所选的调色盘配色。
   - **Export All**：弹出目录选择框，在选定的路径下批量导出左侧列表中存在的所有图片。

## ⚠️ 注意事项与进阶提示

- **高版本加密图档**：部分服务端或高版本的私服端对 `Graphic_*.bin` 进行了 XOR 加密。代码中目前已内置原版解密算法，如果遇到私服二次修改密码，您可能需要修改 `src/core/graphic_info.py` 中的 `self.encrypt_key = b""`。
- **渲染耗时**：如果在点击 `Export All` 导出全量（数万张）图档，应用界面可能会有轻微假死现象，后台由于使用 `numpy` 在进行高强度的矩阵运算，请耐心等待弹出的“Export Complete”提示即可。
- **背景倒置**：工具的像素加载顺序针对 `Pillow` 库已做适配，通常无需手动倒转。如果您发现特定的背景或建筑存在上下颠倒，可在代码中 `graphic_data.py` 底部调整。
